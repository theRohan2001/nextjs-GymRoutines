### GymRoutines is a resistance training workout tracker built as a Progressive Web App. It allows users to log sets, reps, and weights, track workout history, and monitor strength progression over time. Designed for simplicity, offline support, and consistency, the app helps lifters apply progressive overload and stay accountable.
